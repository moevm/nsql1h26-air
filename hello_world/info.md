# Веб-приложение для дыхательных упражнений

Комплексное веб-приложение для управления дыхательными упражнениями и практики их выполнения, созданное с использованием React, TypeScript и ArangoDB.

## Tech Stack

- **Frontend**: React 18, TypeScript, Tailwind CSS, Lucide Icons
- **Backend**: Node.js, Express
- **Database**: ArangoDB (Graph Database)
- **Deployment**: Docker Compose
